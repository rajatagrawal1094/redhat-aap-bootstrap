# Private Automation Hub

Private Automation Hub is used as the governed source for internal automation content.

## Planned Usage

- host the `rajat.platform_bootstrap` collection
- host or reference custom execution environment images
- document curated content sync patterns
- demonstrate namespace and content governance

## Portfolio Message

This project should show that automation content is not just written. It is packaged, versioned, governed, tested, and distributed through enterprise channels.
